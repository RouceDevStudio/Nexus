⚡ Activar Ollama local

      
    

    

      
Con Ollama en tu dispositivo, NEXUS tiene motor de lenguaje local de respaldo cuando Groq no está disponible.

      

        
        
      

      

        

          
1
Instalar Termux
Descárgalo desde F-Droid o tu tienda de apps.

          
2
Instalar y arrancar Ollama
Copia y pega en Termux:
pkg install ollama
OLLAMA_ORIGINS="*" ollama serve

          
3
Descargar modelo
ollama pull llama3.2:1b

          
4
Recarga esta página
NEXUS detectará Ollama automáticamente.

        

        
⚠️ El OLLAMA_ORIGINS="*" es clave para permitir comunicación con la web.

      

      

        

          
1
Descarga Ollama
Entra a ollama.com e instala.

          
2
Descargar modelo e iniciar
ollama pull llama3.2:1b
OLLAMA_ORIGINS="*" ollama serve

          
3
Recarga la página
NEXUS detecta Ollama automáticamente.
